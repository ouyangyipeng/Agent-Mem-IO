## 1. 设计目标

### 1.1 核心约束

| 约束类型 | 具体要求 | 设计影响 |
|----------|----------|----------|
| 内存限制 | 数据集大小的 10%-20% | 必须实现高效的 SSD-offload 机制 |
| 召回率 | Recall@10 ≥ 85% | 图索引必须保持良好的连通性 |
| 读性能 | 高并发 Top-K 查询 | 异步 I/O + 拓扑预取 |
| 写性能 | 高频实时写入不阻塞读 | LSM-Tree 风格写入 |

### 1.2 性能目标

| 指标 | 目标值 | 测量方法 |
|------|--------|----------|
| Recall@10 | ≥ 85% | SIFT1M ground truth |
| 查询 QPS | > 1000 (单线程) | 混合负载基准测试 |
| P99 延迟 | < 10ms | 混合负载基准测试 |
| 内存占用 | < 20% 数据集大小 | cgroups 限制验证 |

## 3. 核心组件设计

### 3.1 Buffer Pool Manager

#### 3.1.1 设计原理

传统 OS Page Cache 对向量检索场景失效的原因：

```
┌─────────────────────────────────────────────────────────────────────────┐
│                    OS Page Cache 失效分析                                │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                         │
│  传统 Page Cache 假设:                                                   │
│  ┌─────────────────────────────────────────────────────────────────┐   │
│  │  1. 时间局部性: 最近访问的页面可能再次访问                       │   │
│  │  2. 空间局部性: 连续的页面可能被顺序访问                         │   │
│  └─────────────────────────────────────────────────────────────────┘   │
│                                                                         │
│  向量图索引的访问模式:                                                   │
│  ┌─────────────────────────────────────────────────────────────────┐   │
│  │  1. 跳跃式访问: 图遍历是随机的，节点 ID 不连续                    │   │
│  │  2. 无顺序预取: "下一跳"由距离计算决定，不可预测                  │   │
│  │  3. 访问频次不均: 枢纽节点被频繁访问，边缘节点很少访问            │   │
│  └─────────────────────────────────────────────────────────────────┘   │
│                                                                         │
│  结果: Page Cache 命中率极低，大量 Page Fault                            │
│                                                                         │
└─────────────────────────────────────────────────────────────────────────┘
```

#### 3.1.3 内存对齐与 O_DIRECT

```cpp
// O_DIRECT 要求:
// 1. 内存缓冲区地址必须对齐到 512 或 4096 字节
// 2. I/O 大小必须是扇区/块大小的整数倍
// 3. 文件偏移必须是扇区/块大小的整数倍

class AlignedBuffer {
public:
    static constexpr size_t ALIGNMENT = 4096;  // 4KB 对齐
    static constexpr size_t PAGE_SIZE = 4096;  // 页大小

    AlignedBuffer(size_t size) {
        // 使用 posix_memalign 分配对齐内存
        void* ptr = nullptr;
        int ret = posix_memalign(&ptr, ALIGNMENT, size);
        if (ret != 0) {
            throw std::bad_alloc();
        }
        data_ = static_cast<char*>(ptr);
        size_ = size;
    }

    ~AlignedBuffer() {
        if (data_) {
            free(data_);
        }
    }

private:
    char* data_;
    size_t size_;
```

```cpp
// 拓扑感知预取器
class TopologyAwarePrefetcher {
public:
    // 在访问节点 N 时，预取其所有邻居
    void prefetch_neighbors(node_id_t node_id, 
                           const GraphIndex& graph,
                           IoUringEngine& io_engine,
                           BufferPool& buffer_pool) {
        // 1. 获取邻居列表
        const auto& neighbors = graph.get_neighbors(node_id);
        
        // 2. 批量提交预取请求
        std::vector<IoRequest> requests;
        for (node_id_t neighbor : neighbors) {
            // 检查是否已在 Buffer Pool 中
            if (!buffer_pool.contains(neighbor)) {
                // 计算向量在 SSD 上的偏移
                off_t offset = compute_vector_offset(neighbor);
                
                // 创建 I/O 请求
                requests.push_back({
                    .fd = vector_file_fd_,
                    .offset = offset,
                    .size = VECTOR_SIZE,  // 128 * 4 = 512 bytes
                    .buffer = buffer_pool.allocate_page(),
                    .user_data = neighbor  // 用于完成回调
                });
            }
        }
        
        // 3. 批量提交到 io_uring
        io_engine.submit_batch(requests);
    }
    
    // 等待预取完成
    void wait_prefetch_complete(IoUringEngine& io_engine,
                                BufferPool& buffer_pool) {
        // 批量收割完成的请求
        auto completed = io_engine.wait_completed_batch();
        
        // 将数据放入 Buffer Pool
        for (const auto& result : completed) {
            buffer_pool.insert(result.user_data, result.buffer);
        }
    }

private:
    int vector_file_fd_;
    static constexpr size_t VECTOR_SIZE = 128 * sizeof(float);  // 512 bytes
};
```

#### 3.3.2 贪心搜索算法

```cpp
// 贪心搜索 (Greedy Search)
std::vector<node_id_t> greedy_search(
    const float* query,
    size_t k,
    const GraphNavData& graph,
    BufferPool& buffer_pool,
    IoUringEngine& io_engine,
    TopologyAwarePrefetcher& prefetcher
) {
    // 优先队列: (距离, 节点ID)
    std::priority_queue<std::pair<float, node_id_t>> candidates;
    std::unordered_set<node_id_t> visited;
    std::vector<node_id_t> results;
    
    // 从入口点开始
    node_id_t current = graph.entry_point;
    candidates.push({0.0f, current});
    visited.insert(current);
    
    while (!candidates.empty() && results.size() < k) {
        // 取出距离最小的候选
        auto [dist, node] = candidates.top();
        candidates.pop();
        
        // 预取邻居节点的向量数据
        prefetcher.prefetch_neighbors(node, graph, io_engine, buffer_pool);
        
        // 获取当前节点的向量 (从 Buffer Pool 或 SSD)
        const float* node_vec = buffer_pool.get_or_load(node);
        
        // 计算距离
        float cur_dist = l2_distance(query, node_vec);
        
        // 加入结果
        results.push_back(node);
        
        // 等待预取完成
        prefetcher.wait_prefetch_complete(io_engine, buffer_pool);
        
        // 遍历邻居
        for (node_id_t neighbor : graph.get_neighbors(node)) {
            if (visited.find(neighbor) == visited.end()) {
                visited.insert(neighbor);
                
                // 获取邻居向量 (应该已在 Buffer Pool 中)
                const float* neighbor_vec = buffer_pool.get(neighbor);
                float neighbor_dist = l2_distance(query, neighbor_vec);
                
                candidates.push({neighbor_dist, neighbor});
            }
        }
    }
    
    return results;
}
```

### 6.3 Recall@10 计算

```python
def compute_recall_at_k(predictions, ground_truth, k=10):
    """
    计算 Recall@K
    
    Args:
        predictions: List of lists, each inner list contains predicted neighbor IDs
        ground_truth: List of lists, each inner list contains true neighbor IDs
        k: Number of neighbors to consider
    
    Returns:
        float: Recall@K score
    """
    total_recall = 0.0
    n = len(predictions)
    
    for i in range(n):
        pred_set = set(predictions[i][:k])
        true_set = set(ground_truth[i][:k])
        
        # Recall = |pred ∩ true| / |true|
        recall = len(pred_set & true_set) / len(true_set)
        total_recall += recall
    
    return total_recall / n
```
