## 🚀 快速开始

```bash
# 构建
mkdir build && cd build
cmake .. && make -j$(nproc)

# 运行测试
./agent_mem_io_tests

# 运行基准测试
./agent_mem_io_benchmark -n 10000 -d 128 -q 100 -k 10
```
